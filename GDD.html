Executive Summary

Game Concept:

The idea of this project is inspired by "Avengers Infinity War". User can select any of avenger's character. After selecting, the user will be enetered in game environment. Main Character will move and will be encountered with enemies at slight distance.It'll kill the enemies and the main objective of the game will be to protect the infinity stones. There are six infinity stones and main character would have to protect all the stones from the main enemy(Thanos) in order to win the game.

Genre:

It's an andventure and Action based game for entertainment purpose for Teen and young audience.

Target Audience:

This game is appropriate for all ages and is simple enough to be enjoyed by causal gamers. As everyone loves the avengers either it is child or young person so, anyone can play it. But specifically the target audience is intendent to be teenagers, most likely to the users over age 10 and young adults. Any male or female can play this game.

Scope:

Our game will be based on creating an action and adventure game with entertainment purposes. The scope of the development of this game is divided in to small cycles which includes the following things:

  1. The first version will start by creating and developing the basic foundations of the game such as the game environment, basic resources e.g., sprites, means of transportation etc. The process will furthur be complexed as the more resources require more sprites, scenes and levels.
  2. The game is single player game, as we only have one main character who will find the infinity stones and will protect it from enemies and there will be a ENEMY who will try to defeat the main character and to steal the infinity stones.
  3. The game will be developed using unity and the scripts will be written in C# Visual studio. All the sprites will be taken from internet.
  4. Character development will not be the part of our game, we will use the resources mainly from internet and the basic functios like running, walking, fighting and finding stones will be done by the main character.
  5. We'll be working mainly with story, levels, object animation, graphics, scripts and game engine facilities.
  6. We will not be working on cartoon making or on web modules i.e., our game will not be deployed on website.

    7. Game Play:

    Objectives:

    The main objective of this game is to protect the infinity stones from enemies. The main character may encounter many hurrdles but his utmost priority is to protect the stones.

    Game Progression:

    The game consists of 3 levels. In each level the main player has to protect the stone from the enemies espacially from the Thanos. There will be difficulties for the main players as the level goes up. Player has guns, power and swords in order to protect the stones. The inital levels are easy and it may take 20-30 minutes to complete.After completing one specific level the player will be directed to next level and some gifts e.g swords, new powers and guns will be given him as reward.
